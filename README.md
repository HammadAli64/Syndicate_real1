# Home (marketing site)

This branch contains only the **Next.js marketing / landing** app under **`frontend/`**.

The full Syndicate stack (Django **Backend/** and dashboard **Frontend-Dashboard/**) lives on the [`main` branch of Syndicate_real1](https://github.com/HammadAli64/Syndicate_real1).

## Run locally

```bash
cd frontend
npm install
npm run dev
```
